form태그는 html에서 사용자가 입력한 값들을 서버로 넘기는 역할을 함
* action 속성 : 폼에 입력된 값들을 전송 받을 서버명 입력
* method 속성 : get / post 방식으로 전송 방식 지정(get방식이 default)
